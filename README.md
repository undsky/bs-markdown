bs-markdown
===========


*基于bootstrap,marked,ace/CodeMirror*


>兼容ie8+,Firefox,Chrome

>预览代码高亮显示

>示例代码嵌入                
            
-------

**highlight by ace**

```javascript
$('#bsmd').bsmd();
```

**highlight by CodeMirror**

```javascript
CodeMirror.autoLoadMode($('#bsmd').bsmd(), 'markdown');
```
